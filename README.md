# How to set up

========================= Command =========================
# First Run
> composer install
> composer update
> php artisan key:generate
> php artisan storage:link
> php artisan serve

# Run Application
> php artisan serve

# Make Controller
> php artisan make:controller <ControllerName>Controller --resource
